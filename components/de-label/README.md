使用：
```
  title：主标题
  fTitle：副标题
  color：主标题颜色
  bgColor：被金色
  size：字体大小（String）
  needBorder：slot区域是否需要上下边框

  <de-label title="布局" fTitle="布局说明文字" color="#ccc" bgColor="#fff" size="28rpx" needBorder="{{ false }}">
    <de-list-group>
      <de-list>这里是插入内容</de-list>
    </de-list-group>
  </de-label>
```
